# Haskell Project

Tetris  

## Usage

```
cabal build
cabal run
```

up arrow: flip  
down arrow: temporarily increase drop speed  
left arrow: shift left  
right arrow: shift right  

Scoring: 50 points per piece + 2^(number of lines cleared) points
